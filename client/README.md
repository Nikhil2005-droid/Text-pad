# Text Pad Frontend

This folder contains the React/Vite frontend for Text Pad.

## Active Source Areas

- `src/components`: shared UI surfaces and layout components
- `src/pages`: route-level screens
- `src/context`: workspace session state
- `src/hooks`: note/code behavior and workspace helpers
- `src/utils`: editor helpers such as auto-replace
- `src/index.css`: shared design system and utility styles

## Commands

- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run preview`

For repository-wide structure and deployment, use the root `README.md` plus `docs/PROJECT-STRUCTURE.md` and `docs/DEPLOYMENT.md`.
